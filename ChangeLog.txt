[ ====== ChangeLog ====== ]

-----------------------------------
 # EggCube stable v1.1 - Client/Server :: https://bitbucket.org/iguanameow/ac_1202/overview
-----------------------------------
 - Restructure layout for ease of use: bin_unix, bin_win32, mods, and client/server .bat/.sh startup files
 - EggCube has its own home folder now located in .assaultcube_eggcube (Linux) and My Documents/AssaultCube_EggCube (Windows Vista/7 or newer)
 - Extra files required by eggcube are now in mods/eggcube

-----------------------------------
 # EggCube stable v1 - Client/Server
-----------------------------------
 - Re-evaluated source code from https://github.com/assaultcube/AC/releases/tag/v1.2.0.2
 - Removed the rest of map restrictions, no more!

-----------------------------
 # EggCube v3 - Client/Server
-----------------------------
 - Ported features from AssaultCube master branch
 - Removed some more map restrictions for online play
 - Enabled blinking text

-----------------------------
 # EggCube v2 - Client/Server
-----------------------------
 - Removed Annoying team switch admin claim cleint server messages
 - Increased spam filter to 50 and 55
 - Added CPistol
 - Adjusted weapon stats

-----------------------------
 # EggCube v1 - Client/Server
-----------------------------
 - Forked from AssaultCube v1.2.0.2 Legacy branch :: https://github.com/assaultcube/AC/tree/legacy1202
 - Default masterserver set to gc.fr.to
 - Remove map restrictions
 - Remove map name restrictions
 - Increased map send and receive size
 - Debug mode enabled for Linux
